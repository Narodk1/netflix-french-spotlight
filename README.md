# 🎬 Netflix French Spotlight

> **Le cinéma français est-il bien représenté sur Netflix ?**  
> Une analyse data complète du catalogue Netflix pour répondre à cette question.

---

## 📊 Résultats clés

| Indicateur | Résultat |
|---|---|
| 🇫🇷 Contenus français | **393 titres** |
| 🌍 Total catalogue Netflix | **8 807 titres** |
| 📉 Part de la France | **4.46%** |
| 🏆 Rang mondial | **5ème pays** |
| 📅 Année record | **2016-2017 (50+ titres)** |
| 🎭 Genre dominant | **International Movies (207)** |

---

## 💡 Insights

- 🇺🇸 Les USA dominent avec **3 689 titres** — presque 10x plus que la France
- 🇰🇷 La Corée du Sud (231 titres) talonne la France malgré une industrie bien plus petite
- 📈 Les ajouts français explosent après **2014**, année d'arrivée de Netflix en France
- 🏷️ Netflix classe les films français comme **"International Movies"** — ils sont donc considérés comme "étrangers" pour l'audience américaine
